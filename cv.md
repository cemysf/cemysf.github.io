# Projects

### Deep learning based people counting using 60-GHz FMCW radar sensor 
#### M.Sc. Graduation Project

This project is about detecting number of persons on indoor scenarios using an FMCW **radar sensor**. I had a range of responsibilites, such as:
- Implementing **signal processing** as radar data processing 
- Applying novel **deep learning** architectures and loss functions for the problem
- Exploring/implementing novel **sensor fusion** architectures with radar and camera data

We have also [published](https://ieeexplore.ieee.org/document/9114871) this work on 2020 IEEE Radar Conference.

### Classification of brain tissues as lesion or healthy by 3D convolutional neural networks
#### B.Sc. Graduation Project

Purpose of the project was to detect and mark ischemic stroke disease in 3D MRI scan volumes.
I have implemented a 3D convolutional neural network model to classify voxels as lesion or
normal and compared its performance against conventional machine learning techniques. For this I  used _tensorFlow_, _scikit-learn_ on **python**.



The dataset is obtained from [2015 Ischemic Stroke Lesion Segmentation (ISLES)
Challenge](http://www.isles-challenge.org/ISLES2015/).  Also, we have published this study as an [academic paper](https://ieeexplore.ieee.org/document/7960524) at IEEE.



![bsc]({{ 'img/cv/bsc.jpg' | relative_url }})
